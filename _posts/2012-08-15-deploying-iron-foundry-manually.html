Over the course of this series I'll dive into setting up independent cloud controllers, DEA (Droplet Execution Agents) and more. For this entry we'll dive straight in and get the cloud controller setup. This will give us a basis in which to launch the other elements of the environment. First, get an instance of Ubuntu Server(we use the LTS version). Once you've downloaded that get it installed, I'll leave that exercise to you. I've setup an instance within my Tier 3 Account. [caption id="attachment_101" align="aligncenter" width="702"] Iron Foundry Cloud Controller Tier 3 Server Instance[/caption] Once that is all setup connect to the instance via SSH.  root@10.81.53.14 would do the trick.  (In my case I'm also behind a VPN, firewall and a whole list of security instruments, so that isn't accessible so don't get any fancy ideas) Generally at this point it is usually a good idea to complete and update and upgrade via apt-get.
